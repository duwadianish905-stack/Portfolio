JavaScript

The programming language that powers interactive web experiences

What is JavaScript?

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. It is a language that is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm.

Alongside HTML and CSS, JavaScript is one of the core technologies of the World Wide Web. JavaScript enables interactive web pages and is an essential part of web applications. The vast majority of websites use it, and major web browsers have a dedicated JavaScript engine to execute it.

JavaScript Basics

JavaScript syntax is the set of rules that define a correctly structured JavaScript program.

// Variables
let message = "Hello, World!";
const pi = 3.14159;
var count = 10;

// Functions
function greet(name) {
  return "Hello, " + name + "!";
}

// Output
console.log(greet("JavaScript Developer"));

JavaScript Data Types

JavaScript is a dynamically typed language, which means you don't have to specify the data type of a variable when you declare it.

Data Type Description Example
String Textual data "Hello World", 'JavaScript'
Number Integer or floating point numbers 42, 3.14
Boolean True or false true, false
Array Ordered list of values [1, 2, 3], ['a', 'b', 'c']
Object Key-value pairs {name: 'John', age: 30}
Null Intentional absence of value null
Undefined Unintentional absence of value undefined

JavaScript in the Browser

JavaScript can manipulate the Document Object Model (DOM) to change document structure, style, and content.

// Changing HTML content
document.getElementById("demo").innerHTML = "Hello JavaScript!";

// Changing CSS styles
document.getElementById("demo").style.color = "red";

// Handling events
document.getElementById("myBtn").onclick = function() {
  alert("Button clicked!");
};

Modern JavaScript (ES6+)

ECMAScript 2015 (ES6) introduced significant updates to JavaScript:

  • Arrow functions: () => {}
  • Classes: class MyClass {}
  • Modules: import/export
  • Template literals: `Hello ${name}`
  • Destructuring: const {prop} = object
  • Promises and async/await for asynchronous operations
  • let and const for block-scoped variables
JavaScript Features
  • Client-side form validation
  • Dynamic content updates
  • Interactive effects and animations
  • API communication (AJAX, Fetch)
  • Browser storage (Cookies, LocalStorage)
  • Asynchronous programming
JavaScript Example
// Calculate factorial
function factorial(n) {
  if (n === 0 || n === 1) {
    return 1;
  } else {
    return n * factorial(n - 1);
  }
}

// Array manipulation
const numbers = [1, 2, 3, 4, 5];
const squares = numbers.map(x => x * x);
console.log(squares); // [1, 4, 9, 16, 25]

// Fetch API example
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
JavaScript Timeline
  • 1995 - Created by Brendan Eich at Netscape
  • 1997 - ECMAScript 1 (First standardization)
  • 2009 - ECMAScript 5 (Major enhancements)
  • 2015 - ECMAScript 6 (Modern JavaScript)
  • Annual releases since 2015 (ES2016, ES2017, etc.)
